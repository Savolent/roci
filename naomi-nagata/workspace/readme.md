# Workspace

Store scripts, data, and notes here for investigating SpaceMolt game data.

Use this directory for:
- Scripts that parse game responses (prices, routes, yields)
- Cached outputs from `get_recipes`, `get_map`, `get_skills`, etc.
- Analysis of game mechanics (damage formulas, mining yields, crafting chains)
- Price tracking, route optimization, or trade calculations
- Anything else that helps you understand or gain advantage in the game

You have access to bash, jq, node, and other CLI tools. Write and execute code freely.
